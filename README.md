## Overview

This repository contains the full computational infrastructure for the paper:

> Rossi, T. (2026). *Modeling the Water–Energy–Food Nexus in Brazilian Periurban
> Watersheds: An Agent-Based Simulation Grounded in Institutional Governance
> Fragmentation and Climate Change Scenarios.* **Water (MDPI)**, Special Issue:
> "The Water–Energy–Food Nexus: Toward Integrated Solutions for Sustainability."
> DOI: [to be inserted after acceptance]

The model simulates interactions among **periurban farmers**, **residential
consumers**, **distributed solar energy prosumers**, and **water resource
managers** in a Brazilian periurban watershed. Agent decisions are shaped by
Brazil's institutionally fragmented WEF governance (ANA / ANEEL / municipal
zoning), generating emergent trade-offs and coordination failures across the
water–energy–food nexus under IPCC AR6 climate scenarios.

## How to Reproduce

### Requirements

| Tool     | Version  | Purpose                        |
|----------|----------|--------------------------------|
| NetLogo  | 6.4.0    | ABM execution                  |
| R        | ≥ 4.3.0  | Statistical analysis & figures |
| renv     | ≥ 1.0.0  | R environment reproducibility  |

### Steps

**1. Clone the repository**
```bash
git clone https://github.com/oThiagoRossi/wef-nexus-abm.git
cd wef-nexus-abm
```

**2. Restore the R environment**
```r
install.packages("renv")
renv::restore()
```

**3. Download raw data**

Large raster files (MapBiomas) are not stored in this repository due to size.
Run the automated download script:
```r
source("analysis/01_preprocess.R")   # downloads and processes all raw data
```
See `data/README_data.md` for manual download instructions and source URLs.

**4. Run the ABM**

Open `model/wef_nexus.nlogo` in NetLogo 6.4.0.
See `model/ODD+D.md` for a full description of agents, parameters, and
scenario configurations.

**5. Generate figures**
```r
source("analysis/04_figures.R")
# Output: outputs/figures/*.pdf (300 DPI)
```
